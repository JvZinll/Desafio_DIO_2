# Xbox Game Pass Sales Dashboard

Projeto de dashboard de vendas criado no Excel.

## Objetivo
Visualizar e analisar vendas de assinaturas (EA Play, Minecraft Season Pass e Xbox Game Pass).

## Ferramentas
- Excel
- Tabelas Dinâmicas
- Segmentações

## Dados
Dados fictícios para fins educacionais.

## Como usar
Utilize os filtros para analisar os resultados.
